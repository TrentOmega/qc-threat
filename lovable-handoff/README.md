# Lovable Handoff Bundle

This folder is a clean attachment bundle for Lovable.

## Folder order

- `01-prompt-and-notes/`
  Contains the Lovable prompt and the source-of-truth website notes.
- `02-reference-pdf/`
  Contains the presentation PDF for visual flow and emphasis reference only.
- `03-selected-assets/`
  Contains only the images and charts referenced by the website notes.

## Usage

Attach the files in this order:

1. `01-prompt-and-notes/qc-threat-website-specifiations.md`
2. `01-prompt-and-notes/qc-threat-website-notes.md`
3. `02-reference-pdf/quantum-threat-bitcoin-bushbash.pdf`
4. the visuals in `03-selected-assets/`

## Note

These are copied into this bundle. The original files remain in their existing repo locations so internal references in the project are not broken.
